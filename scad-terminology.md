# 参数化建模术语表
## 用于 AI 协作建模的语言约定

> 本文档有两个用途：
> 1. 作为 System Prompt 的组成部分，约束 AI 对用户语言的理解
> 2. 作为用户引导文档，帮助没有建模基础的使用者用准确的语言描述意图

---

## 第一部分：空间基础概念

### 坐标系

本工具使用右手坐标系，与 OpenSCAD 一致：

```
Z 轴（↑）= 上下方向，正方向朝上
X 轴（→）= 左右方向，正方向朝右
Y 轴（↗）= 前后方向，正方向朝前（屏幕外）
```

**约定：所有描述方向的语言均以此坐标系为准。**

| 日常语言 | 精确含义 |
|---|---|
| "上方" / "顶部" | Z 轴正方向 (+Z) |
| "下方" / "底部" | Z 轴负方向 (-Z) |
| "左侧" | X 轴负方向 (-X) |
| "右侧" | X 轴正方向 (+X) |
| "前方" | Y 轴正方向 (+Y) |
| "后方" | Y 轴负方向 (-Y) |

---

### 几何元素

**面（Face）**
物体表面上一个连续的平坦或弯曲区域。每个面都有一个法线方向。

**边（Edge）**
两个面相交形成的线段或曲线。边是面和面之间的边界。

**顶点（Vertex）**
三条或以上边交汇的点。

**法线（Normal）**
垂直于面、指向面外侧的方向向量。法线用来精确描述"这个面朝哪个方向"。

```
示例：
  顶面的法线 = [0, 0, 1]（朝上）
  底面的法线 = [0, 0, -1]（朝下）
  右侧面的法线 = [1, 0, 0]（朝右）
```

---

### 面的位置描述

| 用户说 | 精确含义 | 法线方向 |
|---|---|---|
| 顶面 | Z 坐标最大的水平面 | [0, 0, 1] |
| 底面 | Z 坐标最小的水平面 | [0, 0, -1] |
| 前面 | Y 轴正方向的侧面 | [0, 1, 0] |
| 后面 | Y 轴负方向的侧面 | [0, -1, 0] |
| 左侧面 | X 轴负方向的侧面 | [-1, 0, 0] |
| 右侧面 | X 轴正方向的侧面 | [1, 0, 0] |
| 内壁 | 朝向形状内部的面（如孔的内侧） | 法线朝内 |
| 外壁 | 朝向形状外部的面 | 法线朝外 |
| 斜面 | 法线与所有主轴均不平行的面 | 混合方向 |

---

### 边的位置描述

| 用户说 | 精确含义 |
|---|---|
| 顶边 | 位于 Z 坐标最高处的边 |
| 底边 | 位于 Z 坐标最低处的边 |
| 竖边 | 方向平行于 Z 轴的边 |
| 水平边 | 方向平行于 XY 平面的边 |
| 轮廓边 | 形状外轮廓最外侧的边 |
| 内圆边 | 圆孔或圆柱内侧的圆形边 |
| 外圆边 | 圆柱外侧的圆形边 |
| 相交边 | 两个面交界处形成的边 |

## 第二部分：基本形状

### 基本体（Primitive）

构建复杂模型的基础形状。

| 用户说 | 术语 | SCAD 命令 | 关键参数 |
|---|---|---|---|
| 方块 / 长方体 / 盒子 | 立方体 | `cube([x, y, z])` | 长、宽、高 |
| 球 / 圆球 | 球体 | `sphere(r)` | 半径 |
| 圆柱 / 柱子 | 圆柱体 | `cylinder(h, r)` | 高度、半径 |
| 圆锥 | 圆锥体 | `cylinder(h, r1, r2)` | 高度、底半径、顶半径 |
| 棱柱 / 多边形柱 | 多边形柱 | `cylinder` + `$fn` | 边数、高度、半径 |

**参数说明：**
- **半径**：圆心到边缘的距离
- **直径**：圆心到边缘距离的两倍（直径 = 半径 × 2）
- **$fn**：圆形的分段数，值越大越圆滑（建议预览时用 32，输出时用 64 以上）

## 第三部分：布尔操作

布尔操作是用形状对形状进行加减运算，是参数化建模最核心的概念。

### 合并（Union）

**用户常说：** "把这两个拼在一起" / "合并" / "添加"

**含义：** 将两个或多个形状合并成一个整体，重叠部分只保留一份。

```
SCAD：
union() {
  cube([20, 20, 10]);
  cylinder(h=15, r=5);
}
```

---

### 差集（Difference）

**用户常说：** "挖掉" / "打孔" / "切除" / "减去" / "掏空一块"

**含义：** 用第二个形状从第一个形状中减去，留下被切除后的结果。

```
SCAD：
difference() {
  cube([20, 20, 10]);        // 被减的形状（主体）
  cylinder(h=15, r=3);       // 用来挖的形状（工具体）
}
```

**注意：** 第一个列出的是主体，后面的是用来挖的工具体。

---

### 交集（Intersection）

**用户常说：** "只保留重叠的部分" / "取两者共同的区域"

**含义：** 只保留两个形状完全重叠的区域。

```
SCAD：
intersection() {
  cube([20, 20, 20], center=true);
  sphere(r=12);
}
```

---

## 第四部分：变换操作

对形状的位置、角度、大小进行调整。

### 移动（Translate）

**用户常说：** "移动到" / "向某方向偏移" / "平移"

```
SCAD：translate([x, y, z])
示例：translate([10, 0, 5])   // 向右移动 10mm，向上移动 5mm
```

---

### 旋转（Rotate）

**用户常说：** "旋转" / "转动" / "倾斜某个角度"

```
SCAD：rotate([x轴角度, y轴角度, z轴角度])
示例：rotate([0, 0, 45])     // 绕 Z 轴旋转 45 度
      rotate([90, 0, 0])     // 绕 X 轴旋转 90 度（使物体倒向前方）
```

---

### 缩放（Scale）

**用户常说：** "放大" / "缩小" / "拉伸某个方向"

```
SCAD：scale([x, y, z])
示例：scale([2, 1, 1])       // X 方向拉伸为两倍，其他方向不变
      scale([0.5, 0.5, 0.5]) // 整体缩小为原来的一半
```

---

### 镜像（Mirror）

**用户常说：** "镜像" / "翻转复制" / "对称复制"

**含义：** 关于某个平面做翻转复制，生成对称的形状。

```
SCAD：mirror([x, y, z])
示例：mirror([1, 0, 0])      // 关于 YZ 平面（左右）镜像
      mirror([0, 1, 0])      // 关于 XZ 平面（前后）镜像
      mirror([0, 0, 1])      // 关于 XY 平面（上下）镜像
```

---

## 第五部分：成型操作

将 2D 轮廓转化为 3D 形状。

### 线性拉伸（Linear Extrude）

**用户常说：** "拉伸" / "沿某方向延伸" / "做成有厚度的"

**含义：** 将一个 2D 轮廓沿 Z 轴方向拉成 3D 实体。

```
SCAD：linear_extrude(height=10) { 2D 轮廓 }
```

**可选参数：**
- `twist`：拉伸的同时旋转，形成螺旋效果
- `scale`：拉伸的同时缩放截面，形成锥形

---

### 旋转成型（Rotate Extrude）

**用户常说：** "旋转成型" / "车削" / "做成旋转体" / "花瓶形状"

**含义：** 将一个 2D 轮廓绕 Z 轴旋转一圈（或指定角度），形成旋转对称的 3D 实体。

```
SCAD：rotate_extrude(angle=360) { 2D 轮廓 }
```

适合制作：花瓶、碗、杯子、旋转把手、轴承座等。

---

## 第六部分：细节操作

### 倒角（Chamfer）

**用户常说：** "倒角" / "切角" / "斜切边缘" / "把边切成斜的"

**含义：** 将一条锐利的边切成一个斜面。结果是一个平坦的斜切面。

```
视觉效果：
  原来：  ┐        倒角后：  ╗
          │                   │
```

**参数：** 倒角尺寸（斜面的宽度）

---

### 圆角（Fillet）

**用户常说：** "圆角" / "磨圆" / "把边做圆滑" / "去掉锋利的边"

**含义：** 将一条锐利的边替换为一段圆弧过渡。结果是平滑的圆弧面。

```
视觉效果：
  原来：  ┐        圆角后：  ╮
          │                   │
```

**参数：** 圆角半径（圆弧的半径大小）

**倒角 vs 圆角：**
- 倒角 = 斜切，结果是平面
- 圆角 = 磨圆，结果是弧面

---

### 抽壳（Shell）

**用户常说：** "抽壳" / "掏空" / "做成空心的" / "只保留外壳"

**含义：** 将一个实心体变成一个有均匀壁厚的空心壳体，类似把实心的东西掏空，只留一层壳。

**参数：** 壁厚（外壳的厚度）

适合制作：盒子、容器、外壳类零件。

---

### 偏移（Offset）

**用户常说：** "向外扩展" / "向内收缩" / "加厚轮廓" / "放大/缩小轮廓"

**含义（2D）：** 将一个 2D 轮廓向外或向内等距扩展，形成新的轮廓。

**参数：**
- 正值 = 向外扩展
- 负值 = 向内收缩

---

## 第七部分：阵列与复制

### 线性阵列

**用户常说：** "排成一排" / "等间距复制" / "线性排列"

**含义：** 将一个形状沿某个方向等间距复制若干个。

**参数：** 复制数量、间距、方向

```
SCAD：for (i = [0:数量-1]) translate([i * 间距, 0, 0]) { 形状 }
```

---

### 圆形阵列

**用户常说：** "围成一圈" / "圆形排列" / "均匀分布在圆上"

**含义：** 将一个形状沿圆周方向等角度复制若干个。

**参数：** 复制数量、圆的半径、是否均匀分布

---

## 第八部分：位置描述约定

### 描述操作位置时的标准用语

为了让 AI 准确理解操作位置，建议使用以下标准描述方式：

**指定面时：**
```
✓ "在顶面的中心"        → 顶面（法线朝上）的几何中心
✓ "在左侧面靠上的位置"  → 左侧面，Z 方向偏上
✓ "在选中的 face_001 上" → 使用标注编号直接引用
✗ "在那个面上"          → 模糊，无法确定是哪个面
```

**指定边时：**
```
✓ "对顶部四条边做圆角"   → 位于 Z 最高处的四条边
✓ "对 edge_003 做倒角"   → 使用标注编号直接引用
✓ "对圆孔的内边缘做圆角" → 孔的内圆边
✗ "对那条边"             → 模糊，无法确定
```

**指定方向时：**
```
✓ "向上拉伸 10mm"        → 沿 +Z 方向延伸 10mm
✓ "向右偏移 5mm"         → 沿 +X 方向偏移 5mm
✓ "朝法线方向偏移 2mm"   → 沿当前选中面的法线方向
✗ "往那边移一点"         → 无法确定方向和距离
```

## 第九部分：参数化设计约定

参数化设计的核心是用变量代替固定数值，使模型可以通过修改参数快速调整。

### 参数命名约定

```
✓ 使用描述性名称：
  wall_thickness = 3;      // 壁厚
  hole_diameter = 8;       // 孔径
  base_height = 20;        // 底座高度

✗ 避免无意义缩写：
  wt = 3;                  // 无法理解含义
  h = 20;                  // 过于简略
```

### 常用参数类型

| 参数类型 | 示例 | 说明 |
|---|---|---|
| 尺寸参数 | `length = 100` | 长度、宽度、高度等 |
| 半径/直径 | `radius = 10` | 圆形相关尺寸 |
| 厚度 | `thickness = 3` | 壁厚、板厚等 |
| 角度 | `angle = 45` | 旋转或倾斜角度（度数） |
| 数量 | `count = 6` | 阵列数量 |
| 间距 | `spacing = 20` | 阵列间距 |
| 颜色 | `body_color = "SteelBlue"` | 显示颜色 |
| 开关 | `show_holes = true` | 布尔开关 |

## 第十部分：AI 交流指南

### 如何描述一个建模需求

**推荐的描述结构：**

```
1. 主体形状是什么？（基本体 + 尺寸）
2. 需要做哪些加法？（合并哪些形状）
3. 需要做哪些减法？（挖掉哪些部分）
4. 需要哪些细节处理？（倒角、圆角、抽壳）
5. 有哪些参数需要可调？
```

**示例（好的描述）：**
```
主体是一个 60mm × 40mm × 20mm 的长方体。
在顶面中心打一个直径 10mm、深度 15mm 的圆孔。
顶面的四条边做 2mm 圆角。
底面四个角各打一个直径 4mm 的螺丝孔，穿透底部。
壁厚、孔径、整体尺寸都做成可调参数。
```

**示例（需要改进的描述）：**
```
做一个带孔的盒子，边角弄圆一点，
底下有几个小孔用来固定。
```

### 结合点选功能时的描述方式

当你已经点选了某个面或边并获得标注编号时：

```
"对 face_001 中心打一个直径 6mm 的孔，深度 10mm"
"对 edge_003 和 edge_007 做 1.5mm 圆角"
"在 face_002 上添加一个 10mm × 10mm × 5mm 的凸台，居中"
```

使用标注编号是最精确的指代方式，可以消除所有空间歧义。

| 我想做的 | 用这个术语 | 关键参数 |

### 单位约定

本工具默认单位为**毫米（mm）**。

所有尺寸描述如不特别说明，均为毫米。

```
1mm = 0.1cm = 约 0.04 英寸
常见参考：
  信用卡厚度 ≈ 0.8mm
  一元硬币直径 ≈ 25mm
  A4 纸长边 ≈ 297mm
```
